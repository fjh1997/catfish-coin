# Moefish Coin v0.1.9 / 猫鱼币 v0.1.9

## 中文

本版本将英文品牌名从 Catfish 更名为 **Moefish**（对应「猫鱼」，避免与英文鲶鱼及现有 CATFISH 代币混淆），并同步升级桌面客户端与节点二进制。

更新内容：

- 英文名改为 **Moefish Coin**；中文名仍为 **猫鱼币**。
- 桌面客户端可执行文件更名为 `MoefishDero.exe`（数据目录仍为 `%LOCALAPPDATA%\CatfishDeroPublic`，钱包/链数据无需迁移）。
- 保留 v0.1.8 的新图标与隐私币定位文案，以及 v0.1.7 的历史 miniblock 地址校验修复。

使用方式：

1. 下载 `moefish-dero-public-windows.zip`
2. 解压
3. 双击 `MoefishDero.exe`
4. 节点和钱包会自动启动；需要出块时手动点击 `开始挖矿`

## English

This release renames the English brand from Catfish to **Moefish** (matching 猫鱼 / “moe fish”, and avoiding the English word “catfish” and the existing CATFISH token), and ships updated desktop/node binaries.

Changes:

- English name is now **Moefish Coin**; Chinese name remains **猫鱼币**.
- Desktop executable renamed to `MoefishDero.exe` (data directory stays `%LOCALAPPDATA%\CatfishDeroPublic`, so wallet/chain data needs no migration).
- Keeps the v0.1.8 icon/branding refresh and the v0.1.7 historical miniblock address validation fix.

Usage:

1. Download `moefish-dero-public-windows.zip`
2. Extract it
3. Double-click `MoefishDero.exe`
4. The node and wallet start automatically. Click `Start Mining` manually when blocks are needed.
